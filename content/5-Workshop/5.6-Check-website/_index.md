#### Verify Website

After successfully installing and configuring, the next step is to check the website.

1. **Access the created S3 bucket**:
  - Select **Object**.
  - Select the file **index.html**

![index.html](/images/5-Workshop/5.6-Check-website/1.png)


2. Find details of `index.html`:
  - Select **Object URL**.

![Object URL](/images/5-Workshop/5.6-Check-website/2.png)

3. Open URL in a new browser tab:
  - Experience the travel website interface.

![Website](/images/5-Workshop/5.6-Check-website/3.png)
