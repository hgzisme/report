1. Check CloudFront
    - In CloudFront interface, select **ID**.
    ![CloudFront](/images/5-Workshop/5.7-Cloudfront/5.7.3-Deploy-cloudfront/1.png)

    -   At Distribution domain name section:
        -   Ensure CloudFront has finished deploying by checking content in Last modified section.
        -   Select square icon to copy URL.

    ![CloudFront](/images/5-Workshop/5.7-Cloudfront/5.7.3-Deploy-cloudfront/2.png)

    -   Open URL in new browser tab
    ![CloudFront](/images/5-Workshop/5.7-Cloudfront/5.7.3-Deploy-cloudfront/3.png)

    -> Successfully deployed CloudFront
