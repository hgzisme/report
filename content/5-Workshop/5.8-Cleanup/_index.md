#### Cleanup Resources

1.  Empty S3 Bucket
-   Access **AWS S3**.
-   In **Bucket name** list, select the bucket related to the lab.
-   Select **Empty**.
-   In **Empty bucket** page, confirm and select **Empty**.

2. Delete S3 Bucket
-   Access **AWS S3**.
-   Select **S3 bucket** related to the lab.
-   Select **Delete bucket**.

3. Delete CloudFront
-   Access **Amazon CloudFront**.
-   Check the box before **Distributions ID**, on top right - select **Disable**. This process will take a few minutes.
-   After **Disable** is complete, select **Delete**.
