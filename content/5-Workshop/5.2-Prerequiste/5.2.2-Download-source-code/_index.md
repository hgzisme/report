#### Upload Data
Download the source code we will use in this workshop.

Download link: [Source code](https://github.com/hgzisme/workshop-demo/archive/refs/heads/main.zip)

1. In the interface of the newly created S3 bucket.
    - Currently, we see there are no **objects**.
    - Select **Upload** to upload data (the source code downloaded and extracted in the previous step)

![Upload project](/images/5-Workshop/5.2-Prerequisite/5.2.2-Download-source-code/1.png)

2. In the Upload interface
-   Open the window containing the folders and files downloaded and extracted in the step above.
-   Press **Ctrl A** to select all folders and files inside the **workshop-demo-main** directory.
-   Drag all selected folders and files and drop them into the **upload** section of the **S3 Bucket**.

![Upload project](/images/5-Workshop/5.2-Prerequisite/5.2.2-Download-source-code/2.png)
![Upload project](/images/5-Workshop/5.2-Prerequisite/5.2.2-Download-source-code/3.png)

3. Wait for about 5 minutes for the data upload to the S3 bucket to complete.

    -   Interface after upload is finished:
![Upload project](/images/5-Workshop/5.2-Prerequisite/5.2.2-Download-source-code/4.png)

4. View directory in S3 bucket
