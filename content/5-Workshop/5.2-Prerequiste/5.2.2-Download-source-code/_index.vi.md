#### Tải dữ liệu
Tải source code chúng ta sẽ dùng trong workshop lần này

Link tải: [Source code](https://github.com/hgzisme/workshop-demo/archive/refs/heads/main.zip)

1. Trong giao diện S3 bucket vừa tạo.
    - Hiện tại, chúng ta thấy chưa có **object**.
    - Chọn **Upload** để tải dữ liệu (source code đã tải về máy và giải nén)

![Upload project](/images/5-Workshop/5.2-Prerequisite/5.2.2-Download-source-code/1.png)

2. Trong giao diện Upload
-   Bật cửa sổ chứa các folder, file đã down về và giải nén ở bước trên
-   Bấm tổ hợp phím **Ctrl A** để chọn toàn bộ các folder, file có trong thư mục **workshop-demo-main**
-   Kéo toàn bộ các folder, file đã chọn rồi thả vào mục **upload** của **S3 Bucket**

![Upload project](/images/5-Workshop/5.2-Prerequisite/5.2.2-Download-source-code/2.png)
![Upload project](/images/5-Workshop/5.2-Prerequisite/5.2.2-Download-source-code/3.png)

3. Đợi trong khoảng thời gian 5 phút, hoàn thành upload dữ liệu lên S3 bucket.

    -   Giao diện sau khi upload xong:
![Upload project](/images/5-Workshop/5.2-Prerequisite/5.2.2-Download-source-code/4.png)

4. Xem thư mục trong S3 bucket
