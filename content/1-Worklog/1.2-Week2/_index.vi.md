### Kết quả đạt được tuần 2:

* Đã tìm, học và hiểu về VPC 
  * Hiểu thêm về subnet: Public subnet, private subnet
  * Tìm hiểu về các dịch vụ của VPC:
    * Route table
    * Elastic Network Interface (ENI)
    * Elastic IP Address (EIP)
    * Internet Gateway
    * NAT Gateway
  * Tìm hiểu về các tính năng bảo mật của VPC:
    * Security Group
    * Network Access Control List (NACL)
    * Peering Connections (VPC Peering)
    * VPC Flowlogs
    * Transit Gateway
  * Tìm hiểu về các cách kết nối giữa môi trường on-premise/data center và AWS cloud
    * VPN Site-to-Site
    * Direct Connect
  * Tìm hiểu về dịch vụ cân bằng tải Elastic Load Balancer (ELB)
    * Tính năng Sticky Session (Session Affinity)
    * Application Load Balancer
    * Network Load Balancer
    * Classic Load Balancer
    * Gateway Load Balancer

* Hiểu về VPC Endpoint:
  * Hiểu được các khái niệm cơ bản về chúng
  * Hiểu về 2 loại Endpoint:
    * Interface Endpoint
    * Gateway Endpoint
